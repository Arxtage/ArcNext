# v0.7.0

> Released: 2026-03-31

## Highlights

- **Arc-style workspace opening for web links** — new pages now open as docked ArcNext workspaces by default instead of undocked pages, while sites that truly require popup behavior (such as auth, banking, or payment flows) still get lightweight native popups.
- **Search results open into new workspaces** — clicking results on supported search engines now keeps the results page intact and opens the destination in a new workspace, closer to Arc's browsing flow.
- **Smarter workspace navigation** — closing a workspace now returns you to the previously visited workspace instead of jumping to the first one.
- **Media-aware sidebar** — browser workspaces now show playback state in the sidebar and let you mute them directly.

## Improvements

- **Cleaner browser model** — removed the manual undocked-page and dock-back flow in favor of a simpler docked workspace model with popup fallbacks only where sites explicitly need them.
- **Clearer agent status in grouped workspaces** — multi-pane workspace pills now surface per-pane agent activity more accurately.
- **More efficient browser view lifecycle** — inactive browser views suspend more intelligently, warm views age out, and hidden navigations are preserved within the sleep budget for better performance.

## Fixes

- Fixed harmless blocked subframe load failures incorrectly surfacing as browser errors.
- Fixed loopback and localhost favicons leaking into web history.
- Improved browser audio state tracking so sidebar media controls stay in sync.
